SLIDE IMAGES DIRECTORY

Place all slide images in this directory using the following naming convention:
- slide1.png - for Slide 1: Use of Force Training
- slide2.png - for Slide 2: Purpose and Overview
- slide3.png - for Slide 3: Policy Statement
- etc.

Images should ideally be in PNG format with dimensions of 1280x720 pixels (16:9 aspect ratio).

For more detailed instructions, please refer to the README_IMAGES.md file in the parent directory.
