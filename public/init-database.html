Database Initialization Tool

Initialize Database

This tool will initialize your database with all required tables and seed data. Use this if you're experiencing issues with the admin interface or slide editor.

Troubleshooting

If you encounter issues:

  • Make sure your Vercel Postgres database is properly set up
  • Check that the INIT_SECRET environment variable is set in your Vercel project
  • Verify that all database-related environment variables are present

After successful initialization, you should be able to log in with these credentials:

  • Username: admin
  • Password: admin123
